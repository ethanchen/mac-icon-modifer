# mac-icon-modifier

本项目是一个使用 React、Vite 和 TailwindCSS 构建的 Mac 图标修改器。

## 开发&启动

1. 安装依赖

```bash
pnpm install
```

2. 启动开发服务器

```bash
pnpm run dev
```

3. 构建生产环境

```bash
pnpm run build
```

## 技术栈

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

如需修改样式，可直接编辑 `src/index.css` 或使用 Tailwind 实用类。

## 项目结构

- `src/`：React 源代码
- `src/main.jsx`：应用入口
- `src/App.jsx`：主组件
- `src/index.css`：全局样式（已引入 TailwindCSS）
- `vite.config.js`：Vite 配置

默认访问 [http://localhost:5173](http://localhost:5173) 查看项目效果。
